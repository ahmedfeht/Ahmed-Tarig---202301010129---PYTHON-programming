# Week 5 Documentation

## 1. Problem Statement
Create a Python program that automatically calculates a customer's café bill and prints a receipt.

## 2. Inputs
- Customer name
- Coffee quantity
- Tea quantity
- Sandwich quantity

## 3. Outputs
- Customer receipt
- Total bill

## 4. Process Flow
1. Input customer name.
2. Input coffee quantity.
3. Input tea quantity.
4. Input sandwich quantity.
5. Calculate total.
6. Print receipt.

## 5. Constraints
- Quantities must not be negative.
- Prices are fixed.
- Quantities must be integers.

## 6. Decomposition
1. Get customer information.
2. Get item quantities.
3. Calculate total price.
4. Print the receipt.

## 7. Pseudocode

START

Input customer name

Input coffee quantity

Input tea quantity

Input sandwich quantity

Calculate total

Print receipt

END
